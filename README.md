# Academic.kz
Web platform for Kazakhstan applicants (Bachelor/Master/PhD): program matching, application checklist &amp; deadlines, and Bolashak scholarship module.


## Problem
- Applicants struggle with scattered requirements, documents, and deadlines.
- Choosing a suitable program is time-consuming and confusing.
- High risk of missing deadlines or submitting incomplete packages.

## Solution
Academi.kz centralizes admissions prep:
- Program matching (Bachelor/Master/PhD)
- Requirements + deadlines in one place
- Application package checklist with progress tracking
- Bolashak scholarship module

## Key Features (MVP)
- [ ] Programs catalog + smart filters
- [ ] Program details page (requirements, deadlines, documents)
- [ ] Personal application checklist (To Do / In Progress / Done)
- [ ] Deadline tracker (calendar + reminders)
- [ ] Bolashak module (criteria, steps, checklist)
